## Quick Installation

Get Gutter Press up and running in minutes:

```bash
$ git clone https://github.com/BillyRuffian/gutter_press.git my-blog
$ cd my-blog
$ bundle install
$ rails db:migrate
$ bin/dev
```

That's it! Visit `http://localhost:3000` to start creating content.
